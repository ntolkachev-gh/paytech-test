# paytech-test

Spring Boot приложение для работы с PayTech API.

## Локальный запуск

```bash
./gradlew bootRun
```

Приложение будет доступно по адресу: http://localhost:8080

## Переменные окружения

- `PAYTECH_API_TOKEN` - токен API PayTech (по умолчанию используется тестовый токен)
- `PORT` - порт для запуска приложения (по умолчанию 8080)

## Деплой на Railway

### Подготовка

Проект уже настроен для деплоя на Railway. Необходимые файлы:
- `railway.json` - конфигурация Railway
- `nixpacks.toml` - конфигурация сборки через Nixpacks
- `application.yml` - настроен для использования переменной `PORT` из окружения

### Шаги деплоя

1. Создайте проект на [Railway](https://railway.app)
2. Подключите ваш GitHub репозиторий
3. Railway автоматически определит проект как Java приложение
4. Установите переменную окружения `PAYTECH_API_TOKEN` в настройках проекта
5. Railway автоматически соберет и задеплоит приложение

### Переменные окружения на Railway

В настройках проекта Railway добавьте:
- `PAYTECH_API_TOKEN` - ваш токен API PayTech

Переменная `PORT` устанавливается автоматически Railway и не требует ручной настройки.

### Проверка деплоя

После успешного деплоя Railway предоставит публичный URL для вашего приложения.
